PySnake AI

My friend and I developed a version of the popular "Snake" game using Python OOP. Our end goal was to develop an artificial intelligence agent that would control the snake using a Deep-Q algorithm. This project was great in helping me understand some of the more complicated mathematics involved in designing and optimising neural networks.

DLL and BST as Dynamic Set ADT

As a part of my university Algorithms and Data Structures course assessment I have implemented the dynamic set abstract data type as a doubly linked list and a binary search tree in Java and allowed testing this implementation to check its correctness and allow comparing efficiency of both data sets. As this project was focused on complexity, I found it fascinating how different implementation can lead to increased or decreased efficiency and could see the difference by running empirical studies.

Sorting algorithms

As a part of my university Algorithms and Data Structures course assessment I have implemented advanced variations of quicksort as well as insertion and merge sort in Java and performed an analysis of algorithms' performance on different inputs. This project allowed me to develop a much deeper understanding of different approaches to sorting and how the nature of the input data might completely change algorithm's performance.

TCP file transfer in Python

As a part of my university Networks and Operating Systems course assessment, I have collaborated with my colleague on implementing a TCP file transfer server using berkeley sockets API in Python.

Tango with Django project
